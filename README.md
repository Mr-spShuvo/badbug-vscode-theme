## Installation

1. Open the extensions sidebar on Visual Studio Code
2. Search for **BADBUG Theme**
3. Click **Install** to install it.
4. Click **Reload** to reload the your editor
5. File > Preferences > Color Theme > **BADBUG - `<Theme>`**
